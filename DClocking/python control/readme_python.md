# 概要

## 信号类型

信号分为两大类，物理信号与布尔信号。

所有物理信号均以16位有符号整数表示，范围为-32768至32767，又根据其含义分为以下几类：
- 电平信号
- 相位信号
- 相位差分信号

各个模块的接口有对应信号类型，物理信号与布尔信号间，以及不同类型的物理信号间均不可直接连接。

例如：
- 标注为“1bit布尔值”的端口不能与标注为“任意物理信号”的端口连接
- 标注为“相位信号”的端口不能与标注为“相位差分信号”的端口连接
- 标注为“任意物理信号”的端口可以与标注为“电平信号”、“相位信号”或“相位差分信号”的端口连接

通过连接线粗细区分物理信号与布尔信号，通过颜色区分不同类型的物理信号。

例如：
- 从物理信号端口引出的连接线为粗线，从布尔信号端口引出的连接线为细线
- “相位信号”端口标记为A色，“相位差分信号”端口标记为B色，从中引出的线也为对应颜色
- “电平信号/相位信号”端口标记为对应颜色拼接的双色色块，从中引出的线也为对应颜色
- 从“电平信号/相位信号”端口引出的线连接至“相位信号”端口后，该线变为单一颜色

以上仅为设想，可自行决定区分方式。

## 参数设定

参数设定格式：
<模块变量名>.write(<地址/参数名>, <值>)

```python
pid = module.ModulePID(bus_inst, "PIDC")
pid.write(0, 16384)
pid.write("gain_p", 16384)  
pid.write("saturation_gain", 10) # 10dB
```

参数读取格式：
<模块变量名>.read(<地址/参数名>)

```python
...
pid.read("gain_p") # 16384
pid.read("saturation_gain") # 10
```

部分间接参数在读取时可能返回无穷大np.inf。

部分位宽为1的参数，仅在上升沿或下降沿有效，对应格式：
<模块变量名>.flip_on(<地址/参数名>)
<模块变量名>.flip_off(<地址/参数名>)

```python
sclofsm = module.ModuleSCLOFSM(bus_inst, "SCLO")
sclofsm.flip_on("lock") # 在lock参数产生一个上升沿
sclofsm.flip_off("lock") # 在lock参数产生一个下降沿
```

## 模块连接

端口连接格式：

router.set_routing(<目标端口名>, <源端口名>)

router.upload()

```python
router.set_routing("TRI_IN", "ACC_OUT")
router.set_routing("OUTPUT_A", "TRI_SIN")
router.upload() # 一次同时上传所有连接设定
```

特殊端口：
- VOID：通往该端口的16bit物理信号被忽略，来自该端口的信号恒为0，用于表示“断开”状态。
- VOID_BOOL：同上，但适用于布尔信号。
- LOW：来自该端口的布尔信号恒为0。
- HIGH：来自该端口的布尔信号恒为1。

对于布尔信号端口，允许用户直接连接至LOW或HIGH端口。

### 端口

- 误差信号输入（电平信号/相位信号）

输出：
- 控制信号输出（电平信号/相位差分信号）

端口名：
|标识符|自动开关指令|误差信号输入|控制信号输出|
|----|----|----|----|
|PIDC|PID_RESET|PID_IN|PID_OUT|
|PID2|PID2_RESET|PID2_IN|PID2_OUT|


### 直接参数

|地址|参数名|位宽|范围|含义|备注|
|----|----|----|----|----|----|
|0|gain_p|24|-2^23 ~ 2^23-1|P通道增益系数|线性增益系数。取对数后才可转换为分贝值。可通过间接参数直接设定分贝值|
|1|gain_i|32|-2^31 ~ 2^31-1|I通道增益系数|同上|
|2|gain_d|24|-2^23 ~ 2^23-1|D通道增益系数|同上|
|3|setpoint|16|-32768 ~ 32767|设定值|物理信号|
|4|limit_integral|16|0 ~ 32767|积分限幅值|物理信号|
|5|limit_sum|16|0~ 32767|输出限幅值|物理信号|
|6|leak_digit|32|无效|积分泄漏控制参数|仅推荐通过间接参数修改
|7|enable_auto_reset|1|0 / 1|自动控制使能|设为1时，模块接受其它状态机模块的开关指令。设为0时，开关指令被忽略|

### 间接参数

|参数名|范围|含义|备注|
|----|----|----|----|
|overall_gain|实数，单位为dB|整体增益|调节P通道增益，并等比例缩放I、D通道增益。若缩放后任一通道增益超出范围则报错。若初始P通道增益为0（缩放比例为无穷大），则仅调节P通道增益|
|pi_corner|非负实数，单位为Hz|PI交点频率|调节I通道增益。使得PI增益曲线交点位于指定频率。若调节后I通道增益超出范围则报错|
|pd_corner|非负实数，单位为Hz|PD交点频率|调节D通道增益。使得PD增益曲线交点位于指定频率。若调节后D通道增益超出范围则报错|
|saturation_gain|实数，单位为dB|饱和增益|调节泄漏系数，使得积分通道在设定增益处饱和。若饱和增益过大，则设定为无饱和。若饱和增益过小，则设定为最大泄漏。若I通道增益为0，则报错|
|saturation_turning_frequency|非负实数，单位为Hz|饱和拐点频率|调节泄漏系数，使得积分通道在设定频率处饱和。若饱和频率过小或为0，则设定为无饱和。若饱和频率过大，则设定为最大泄漏|

### 可视化

在自由模式下，允许用户调整除了leak_digit以外的所有直接和间接参数。

绘制PID控制器的增益曲线：
- 一个坐标系，横坐标为对数频率（Hz），纵坐标为增益（dB）。
- 根据overall_gain绘制比例通道的响应，斜率为0，截距即为overall_gain。
- 根据pi_corner绘制积分通道的响应，斜率为-20dB/十倍频程，在pi_corner处与比例通道响应曲线相交。
- 根据pd_corner绘制微分通道的响应，斜率为+20dB/十倍频程，在pd_corner处与比例通道响应曲线相交。
- 根据saturation_turning_frequency绘制积分饱和响应，斜率为0，在saturation_turning_frequency处与积分通道响应曲线相交。
- 用户可以通过调整参数，观察增益曲线的变化。
- 注意根据gain_p、gain_i、gain_d的实际值，上述参数可能出现0或者无穷大的特殊情况，绘图时需要进行相应处理。

### 特殊方法

方法design_lowpass用于设计一个FIR低通滤波器并立即载入生效。

定义：
```python
def design_lowpass(self, freq_pass, freq_stop, freq_sample, weight = 1, taps = 64):
```

参数说明：
- freq_pass：通带截止频率，单位为Hz
- freq_stop：阻带截止频率，单位为Hz
- freq_sample：采样频率，单位为Hz
- weight：阻带权重，默认为1
- taps：滤波器抽头数，默认为64，仅支持16、32或64抽头

用法：
```python
fir = module.ModuleFIRFilter(bus_inst, "FIRF")
fir.design_lowpass(1e6, 10e6, 250e6) # 设计一个通带截止频率为1MHz，阻带截止频率为10MHz的64抽头低通滤波器
```

### 特殊方法

方法design_lowpass用于设计一个IIR低通滤波器并立即载入生效。

定义：
```python
def design_lowpass(self, filter_type, freq_pass, freq_sample):
```

参数说明：
- filter_type：滤波器类型，"butter"为Butterworth滤波器，"cheby1"为Chebyshev I型滤波器，"cheby2"为Chebyshev II型滤波器，"ellip"为椭圆滤波器
- freq_pass：通带截止频率，单位为Hz
- freq_sample：采样频率，单位为Hz

用法：
```python
iir = module.ModuleIIRFilter(bus_inst, "IIRF")
iir.design_lowpass("ellip", 10e6, 250e6) # 设计一个通带截止频率为10MHz的椭圆低通滤波器
```
